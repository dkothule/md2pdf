# md2pdf Test Samples

This folder contains repo-only test content for validating Mermaid rendering behavior.

These files are intentionally not included in the published npm package.

## Quick run

```bash
md2pdf ./tests/architecture-smoke-test.md
```

## Full Mermaid sample suite

```bash
md2pdf ./tests/samples/mermaid-all-diagram-types.md --keep-mermaid-assets
```
